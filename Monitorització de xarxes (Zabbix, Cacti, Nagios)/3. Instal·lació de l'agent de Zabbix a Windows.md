# Instal·lació i configuració de l'agent de Zabbix en Windows
---

## 1. Instal·lació de l'agent a la màquina client

### Pas 1: Benvinguda i llicència
Iniciem l'execució de l'instal·lador oficial de Zabbix. Acceptem els termes de la llicència per poder continuar amb l'assistent.
![Pas 1](img/ins_win/1rpas.png)
![Pas 2](img/ins_win/2npas.png)
![Pas 3](img/ins_win/3rpas.png)

### Pas 2: Configuració del servidor (Zabbix Server)
Configurem els detalls de la connexió amb el servidor:
- **Host name**: El nom que identificarà aquest equip.
- **Zabbix server IP**: L'adreça del servidor que recollirà les dades.
- **Listen port**: Port de l'agent (per defecte 10050).

![Servidor Zabbix](img/ins_win/Server.png)
![IP Host Zabbix](img/ins_win/Ip_host_za.png)
![Hostname configuració](img/ins_win/Hostname_conf.png)
![IP configuració](img/ins_win/Ip_conf.png)

### Pas 3: Selecció de ruta i components
Triem el directori on s'instal·laran els fitxers i els components de l'agent.
![Ruta 1](img/ins_win/Ruta2.png)
![Ruta 2](img/ins_win/Ruta_config.png)
![Part de l'agent](img/ins_win/Part_agent.png)
![Opcions](img/ins_win/Opcio.png)
![Més opcions de l'agent](img/ins_win/Opcions_agent.png)

### Pas 4: Procés d'instal·lació i finalització
L'assistent copiarà els fitxers i registrarà el servei al sistema Windows.
![Instal·lant](img/ins_win/Instal·lar.png)
![Finalitzat](img/ins_win/Finish.png)

## 2. Configuració del Firewall de Windows

Cal obrir el port **10050/TCP** per permetre que el servidor es pugui comunicar amb l'agent.

### Pas 1: Crear nova regla
Accedim al Firewall de Windows Defender i creem una nova regla d'entrada.
![Comanda Firewall](img/ins_win/Comanda%20Firewall.png)
![Nova regla d'entrada](img/ins_win/FW_ENT_1.png)

### Pas 2: Paràmetres de la regla
Configurem el port, el nom i els perfils de xarxa.
![Nom de la regla](img/ins_win/Nom_regla.png)
![Ports](img/ins_win/Ports.png)
![On s'aplica](img/ins_win/On_saplica.png)
![Permetre connexió](img/ins_win/Permetre3.png)
