# Guia de configuracio de Zabbix Agent 2 amb PSK a Windows

Aquest document mostra, pas a pas, com donar d'alta un host Windows a Zabbix utilitzant xifrat PSK (Pre-Shared Key).

## Objectiu

Configurar un equip Windows anomenat `server02` perque es connecti al servidor Zabbix `192.168.56.101` mitjancant `Zabbix Agent 2` amb autenticacio PSK.

## Dades utilitzades a l'exemple

- Hostname: `server02`
- Servidor Zabbix: `192.168.56.101`
- Port de l'agent: `10050`
- Grup d'equips: `Windows Server`
- Plantilla: `Windows by Zabbix agent active`

## Pas 1. Generar la clau PSK al servidor Zabbix

Genera una clau PSK de 32 bytes en format hexadecimal des del servidor Zabbix:

```bash
openssl rand -hex 32
```

Aquesta clau es fara servir tant a la instal.lacio de l'agent com a la configuracio del host dins de Zabbix.

![Generacio de la PSK](img/PSK-Win/PSK2.png)

## Pas 2. Configurar l'agent Windows durant la instal.lacio

Durant la instal.lacio de `Zabbix Agent 2`, omple les dades basiques del host:

- Introdueix el nom de l'equip a `Host name`
- Indica la IP o DNS del servidor Zabbix
- Deixa el port de l'agent a `10050`
- Introdueix el servidor o proxy per a `active checks`
- Marca l'opcio `Enable PSK`

![Configuracio inicial del servei](img/PSK-Win/PSK1.png)

## Pas 3. Introduir la identitat i la clau PSK

Quan l'instal.lador et demani la configuracio PSK:

- A `Pre-shared key identity`, escriu una identitat descriptiva, en aquest exemple `server02`
- A `Pre-shared key value`, enganxa la clau generada amb `openssl`

Despres continua amb `Next`.

![Configuracio de la PSK a l'agent](img/PSK-Win/PSK3.png)

## Pas 4. Crear un grup d'equips a Zabbix

Des de la interfície web de Zabbix, ves a:

`Recopilacion de datos` -> `Grupos de equipos`

A continuacio, prem `Crear grupo de equipos`.

![Acces a grups d'equips](img/PSK-Win/PSK4.png)

## Pas 5. Afegir el grup per als hosts Windows

Crea un grup nou amb el nom `Windows Server`.

Aquest grup servira per classificar els equips Windows que monitoritzaras.

![Creacio del grup Windows Server](img/PSK-Win/PSK5.png)

## Pas 6. Verificar que el grup s'ha creat

Comprova que el grup `Windows Server` apareix dins del llistat de grups disponibles.

![Grup creat correctament](img/PSK-Win/PSK6.png)

## Pas 7. Crear el nou host

Ara ves a:

`Recopilacion de datos` -> `Equipos`

I prem `Crear equipo`.

![Acces a la creacio del host](img/PSK-Win/PSK7.png)

## Pas 8. Omplir la configuracio basica del host

A la pestanya `Equipo`, configura:

- `Nombre de equipo`: `server02`
- Plantilla: `Windows by Zabbix agent active`
- Grup d'equips: `Windows Server`
- Interficie de l'agent amb la IP del host, en l'exemple `192.168.56.104`
- Port: `10050`

Quan estigui tot omplert, encara no guardis: primer cal revisar la pestanya de xifrat.

![Configuracio general del host](img/PSK-Win/PSK8.png)

## Pas 9. Obrir la pestanya de xifrat

Dins del formulari del host, entra a la pestanya `Cifrado`.

![Pestanya de xifrat](img/PSK-Win/PSK9.png)

## Pas 10. Configurar el xifrat PSK al host de Zabbix

A la pestanya `Cifrado`:

- A `Conexiones al host`, selecciona `PSK`
- A `Conexiones desde el host`, marca tambe `PSK`
- A `Identidad PSK`, introdueix `server02`
- A `PSK`, enganxa exactament la mateixa clau generada al pas 1

Despres prem `Agregar`.

![Configuracio PSK al host de Zabbix](img/PSK-Win/PSK10.png)

## Pas 11. Filtrar i comprovar el nou host

Un cop creat el host, pots buscar-lo a `Datos mas recientes` filtrant per `server02` i aplicant el filtre.

Aquest pas et permet validar rapidament que el host ja existeix i esta enviant dades o intentant enviar-ne.

![Filtre del host a dades recents](img/PSK-Win/PSK11.png)

## Pas 12. Validar que arriben les metriques

Si la configuracio es correcta, veuras els items del host `server02` dins de `Datos mas recientes`, com per exemple:

- CPU
- memoria
- processos
- sistema
- filesystem

Aixo confirma que el `Zabbix Agent 2` esta registrat i reportant informacio al servidor.

![Metriques del host a Zabbix](img/PSK-Win/PSK12.png)

## Resum rapid

1. Genera la clau PSK al servidor Zabbix.
2. Instal.la `Zabbix Agent 2` a Windows i activa `Enable PSK`.
3. Introdueix la identitat i la clau PSK a l'instal.lador.
4. Crea el grup `Windows Server`.
5. Dona d'alta el host `server02`.
6. Assigna la plantilla `Windows by Zabbix agent active`.
7. Configura el xifrat PSK amb la mateixa identitat i clau.
8. Verifica a `Datos mas recientes` que entren les metriques.
