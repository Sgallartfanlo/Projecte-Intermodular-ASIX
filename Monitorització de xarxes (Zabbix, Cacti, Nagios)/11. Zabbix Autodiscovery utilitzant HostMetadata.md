# Gestió i HMD-registració de Hosts a Zabbix

## Fase 1: Eliminació d’un Host

### Pas 1: Accedir a la llista d’equips

![Llista equips](img/HMD/HM14.png)

* Anar a:

  * **Recopilació de dades → Equips**

Aquí es mostren tots els dispositius monitoritzats.

---

### Pas 2: Seleccionar el dispositiu

![Seleccionar host](img/HMD/HM14.png)

* Marcar el host que vols eliminar (per exemple `srvlinux01`)

---

### Pas 3: Eliminar el host

![Eliminar host](img/HMD/HM14.png)

* Fer clic a **Eliminar**
* Confirmar l’acció

---

### Pas 4: Verificar eliminació

* El host desapareix de la llista
* Ja no es monitoritza ni envia dades

---

## Important: HMD-registre

Si tens configurat l’HMD-registre, el host tornarà a aparèixer HMDmàticament.

Per evitar-ho:

### Opció 1: Aturar l’agent al host

```bash
systemctl stop zabbix-agent
```

---

### Opció 2: Eliminar el HostMetadata

![HostMetadata Linux](img/HMD/HM3.png)

Editar configuració:

```bash
nano /etc/zabbix/zabbix_agentd.conf
```

Eliminar o comentar:

```bash
HostMetadata=Linux
```

Reiniciar:

```bash
systemctl restart zabbix-agent
```

---

## Fase 2: Configuració de l’Agent Zabbix

### Pas 1: Editar configuració

![Editar configuració agent](img/HMD/HM1.png)

```bash
nano /etc/zabbix/zabbix_agentd.conf
```

---

### Pas 2: Configurar connexió

![Configuració agent](img/HMD/HM2.png)

```bash
ServerActive=192.168.56.101
Hostname=srvlinux01
```

---

### Pas 3: Afegir metadades

![HostMetadata Linux](img/HMD/HM3.png)

```bash
HostMetadata=Linux
```

---

### Pas 4: Reiniciar servei

![Restart servei](img/HMD/HM4.png)

```bash
systemctl restart zabbix-agent.service
systemctl status zabbix-agent.service
```
