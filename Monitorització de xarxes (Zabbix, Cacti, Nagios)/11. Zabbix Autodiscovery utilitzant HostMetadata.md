# Auto-registració i Descobriment de Hosts a Zabbix

Aquest document descriu el procés complet per configurar l’autoregistració automàtica de hosts Linux a Zabbix mitjançant **HostMetadata** i el descobriment de xarxa.

---

## Fase 0: Eliminació del host existent

![Configuració agent](img/HMD/HM1.png)

Abans de començar, eliminem el host `srvlinux01` per comprovar que es tornarà a crear automàticament.

Passos:

* Anar a **Recopilación de datos → Equipos**
* Seleccionar el host
* Fer clic a **Eliminar**

---

## Fase 1: Configuració de l’Agent Zabbix

### Pas 1: Editar el fitxer de configuració

![Configuració agent](img/HMD/HM2.png)

```bash
nano /etc/zabbix/zabbix_agentd.conf
```

Aquest fitxer controla la comunicació entre l’agent i el servidor.

---

### Pas 2: Configurar connexió i identificació

![Configuració agent](img/HMD/HM3.png)

```bash
ServerActive=192.168.56.101
Hostname=srvlinux01
```

* **ServerActive**: IP del servidor Zabbix
* **Hostname**: Nom del host dins Zabbix

---

### Pas 3: Configurar HostMetadata

![Configuració agent](img/HMD/HM5.png)

```bash
HostMetadata=Linux
```

Aquesta metadada s’utilitza per aplicar regles d’autoregistració.

---

### Pas 4: Reiniciar el servei
![Configuració agent](img/HMD/HM4.png)


```bash
systemctl restart zabbix-agent.service
systemctl status zabbix-agent.service
```

Comprovem que l’estat sigui **active (running)**.

---

### Pas 5: Comprovar logs

![Configuració agent](img/HMD/HM6.png)

```bash
tail -f /var/log/zabbix/zabbix_agentd.log
```

Permet detectar errors i verificar la connexió amb el servidor.

---

## Fase 2: Configuració del Descobriment

### Pas 1: Accedir a descobriment

![Configuració agent](img/HMD/HM7.png)

Ruta:

* **Recopilación de datos → Descubrimiento**

---

### Pas 2: Crear regla de descobriment

![Configuració agent](img/HMD/HM8.png)

Configurem:

* **Nom**: Local network
* **Rang IP**: `192.168.56.1-254`
* **Interval**: 1h
* **Tipus**: Agente Zabbix

---

### Pas 3: Configuració avançada

![Configuració agent](img/HMD/HM9.png)

* Criteri d’unicitat: **Direcció IP**
* Nom de host: **DNS**
* Estat: **Activat**

---

## Fase 3: Configuració d’Autoregistració

### Pas 1: Crear acció

![Configuració agent](img/HMD/HM10.png)

Ruta:

* **Alertas → Acciones → Autoregistro**

Nom:

```
AutoRegistrar Servidor Linux
```

---

### Pas 2: Afegir condició

![Configuració agent](img/HMD/HM11.png)

Condició:

```
Metadata contiene Linux
```

Això fa que només s’apliqui a equips Linux.

---

### Pas 3: Configurar operacions

![Configuració agent](img/HMD/HM12.png)

Afegim:

* Afegir host
* Afegir a grup: `Linux servers`
* Vincular plantilles:

  * `Linux by Zabbix agent`
  * `ICMP Ping`
* Activar host

---

### Pas 4: Verificar acció

![Configuració agent](img/HMD/HM13.png)

Comprovem que l’acció està **Activada**.

---

## Fase 4: Verificació del sistema

### Pas 1: Estat del descobriment

![Configuració agent](img/HMD/HM14.png)

Podem veure:

* IP detectada
* Host associat (`srvlinux01`)

---

### Pas 2: Confirmar host creat

A la secció **Equipos** comprovem que:

* El host s’ha creat automàticament
* Té plantilles assignades
* Estat: **Activat**
