# HMD-registració i Descobriment de Hosts a Zabbix

## Fase 1: Configuració de l’Agent Zabbix (Linux)

### Pas 1: Editant la configuració de l’agent

![Editar configuració agent](img/HMD/HM1.png)

* Obrint el fitxer de configuració:

```bash
nano /etc/zabbix/zabbix_agentd.conf
```

### Pas 2: Configurant el servidor i hostname

![Configuració agent](img/HMD/HM2.png)

* Afegir la IP del servidor Zabbix:

```bash
ServerActive=192.168.56.101
```

* Definir el nom del host (ha de coincidir amb Zabbix):

```bash
Hostname=srvlinux01
```

### Pas 3: Afegint metadades per HMDregistre

![HostMetadata Linux](img/HMD/HM3.png)

* Afegir la línia:

```bash
HostMetadata=Linux
```

Això permetrà que Zabbix identifiqui el tipus de sistema per aplicar accions HMDmàtiques.

### Pas 4: Reiniciant el servei

![Restart servei](img/HMD/HM4.png)

```bash
systemctl restart zabbix-agent.service
systemctl status zabbix-agent.service
```

* Verificar que l’estat sigui **active (running)**

## Fase 2: Creació de Regla de Descobriment

### Pas 1: Accedint a Descobriment

![Menú descobriment](img/HMD/HM5.png)

* Anar a:

  * **Recopilació de dades → Descobriment**

### Pas 2: Configurant la regla

![Regla descobriment](img/HMD/HM6.png)

* **Nom:** Local network
* **Rang IP:** 192.168.56.1-254
* **Interval:** 1h
* **Comprovació:** Agente Zabbix

### Pas 3: Activant la regla

![Regla activada](img/HMD/HM7.png)

* Assegurar que l’estat sigui **Activado**

---

## Fase 3: Configuració d’HMD-registració

### Pas 1: Creant una nova acció

![Crear acció](img/HMD/HM8.png)

* Anar a:

  * **Alertes → Accions → HMDregistre**
* Crear nova acció

### Pas 2: Definint la condició

![Condició Linux](img/HMD/HM9.png)

* **Tipus:** Metadatos de equipo
* **Operador:** contiene
* **Valor:** Linux

### Pas 3: Confirmació de la condició

![Condició creada](img/HMD/HM10.png)

* Verificar que la condició s’ha afegit correctament

## Fase 4: Configuració d’Operacions

### Pas 1: Afegint operacions

![Operacions](img/HMD/HM11.png)

Afegir les següents accions:

* Afegir equip
* Afegir a grup: **Linux servers**
* Vincular plantilla: **Linux by Zabbix agent active**
* Vincular plantilla: **ICMP Ping**
* Activar equip

### Pas 2: Confirmació de l’acció

![Acció creada](img/HMD/HM12.png)

* Verificar que l’acció està **Activada**

![Logs agent](img/HMD/HM15.png)

```bash
tail -f /var/log/zabbix/zabbix_agentd.log

```

* Els hosts es detecten HMDmàticament
* Es registren sense intervenció manual
* S’assignen plantilles i grups HMDmàticament
* Comencen a enviar dades immediatament
