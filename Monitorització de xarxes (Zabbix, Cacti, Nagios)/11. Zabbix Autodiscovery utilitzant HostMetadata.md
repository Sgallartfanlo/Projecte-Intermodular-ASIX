# Auto-registració i Gestió de Hosts a Zabbix

Aquest document descriu el procés complet de configuració de l’auto-registració de hosts Linux a Zabbix.

**Nota:** Els hosts es poden eliminar des de *Recopilació de dades → Equips*. Cal tenir en compte que, si l’auto-registració està activa, el dispositiu pot tornar a aparèixer si l’agent continua funcionant.

---

## Fase 1: Configuració de l’Agent Zabbix (Client)

### Pas 1: Editar el fitxer de configuració

![Editar configuració agent](img/HMD/HM1.png)

```bash id="9bld72"
nano /etc/zabbix/zabbix_agentd.conf
```

Aquest fitxer controla la comunicació entre el client i el servidor Zabbix.

---

### Pas 2: Configurar connexió amb el servidor

```bash id="5kp7pn"
ServerActive=192.168.56.101
Hostname=srvlinux01
```

* **ServerActive**: IP del servidor Zabbix
* **Hostname**: Nom del host

---

### Pas 3: Afegir metadades per autoregistre

![HostMetadata Linux](img/HMD/HM3.png)

```bash id="y7o6nb"
HostMetadata=Linux
```

Aquest valor permet aplicar regles automàtiques.

---

### Pas 4: Reiniciar el servei

![Restart servei](img/HMD/HM4.png)

```bash id="r4gzk5"
systemctl restart zabbix-agent.service
systemctl status zabbix-agent.service
```
