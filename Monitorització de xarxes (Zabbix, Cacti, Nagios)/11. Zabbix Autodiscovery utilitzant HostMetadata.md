# Autoregistre i Descobriment de Servidors Linux a Zabbix

- Descobrir equips a la xarxa
- Autoregistrar servidors Linux
- Assignar-los plantilles i grups automàticament
- Validar que el procés funciona

## 2. Edició del fitxer de configuració del Zabbix Agent
![img](img/HMD/HM2.png)

 Obrim el fitxer de configuració del Zabbix Agent.

Codi:
```bash
nano /etc/zabbix/zabbix_agentd.conf
```

## 3. Configuració del Zabbix Agent
 Configuració essencial del Zabbix Agent.

Codi:
```conf
ServerActive=192.168.56.101
Hostname=srvlunix01
HostMetadata=Linux
```

## 4. Reinici del servei Zabbix Agent
 Reiniciem el servei i comprovem l’estat.

Codi:
```bash

```
