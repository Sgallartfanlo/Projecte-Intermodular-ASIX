# Auto-registració i Gestió de Hosts a Zabbix

## Fase 0: Eliminació d’un Host

### Pas 1: Accedir a la llista d’equips

![Seleccionar host](img/HMD/HM14.png)

* Anar a:

  * **Recopilació de dades → Equips**

---

### Pas 2: Seleccionar el host

* Marcar el dispositiu que es vol eliminar (ex: `srvlinux01`)

---

### Pas 3: Eliminar el host

* Clicar **Eliminar**
* Confirmar l’acció

---

### Pas 4: Verificar eliminació

* El host desapareix de la llista
* Ja no es monitoritza

---

## Fase 1: Configuració de l’Agent Zabbix (Client)

### Pas 1: Editar el fitxer de configuració

![Editar configuració agent](img/HMD/HM1.png)

```bash
nano /etc/zabbix/zabbix_agentd.conf
```

---

### Pas 2: Configurar connexió amb el servidor

```bash
ServerActive=192.168.56.101
Hostname=srvlinux01
```

---

### Pas 3: Afegir metadades per autoregistre

![HostMetadata Linux](img/HMD/HM3.png)

```bash
HostMetadata=Linux
```

---

### Pas 4: Reiniciar el servei

![Restart servei](img/HMD/HM4.png)

```bash
systemctl restart zabbix-agent.service
systemctl status zabbix-agent.service
```

---

## Important: Evitar re-registre automàtic

### Opció 1: Aturar agent

```bash
systemctl stop zabbix-agent
```

### Opció 2: Eliminar metadata

```bash
nano /etc/zabbix/zabbix_agentd.conf
```

Eliminar:

```bash
HostMetadata=Linux
```

Reiniciar:

```bash
systemctl restart zabbix-agent
```
