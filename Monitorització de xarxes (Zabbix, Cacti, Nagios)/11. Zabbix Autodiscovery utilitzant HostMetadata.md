# Auto-registració i Gestió de Hosts a Zabbix

## Fase 1: Configuració de l’Agent Zabbix

### Pas 1: Editar la configuració de l’agent

![Editar configuració agent](img/HMD/HM1.png)

```bash
nano /etc/zabbix/zabbix_agentd.conf
```

Aquest fitxer controla la comunicació entre el host i el servidor Zabbix.

---

### Pas 2: Configurar la connexió amb el servidor

![Configuració agent](img/HMD/HM2.png)

```bash
ServerActive=192.168.56.101
Hostname=srvlinux01
```

* **ServerActive:** IP del servidor Zabbix
* **Hostname:** nom amb què el dispositiu es registrarà a Zabbix

---

### Pas 3: Afegir metadades per a l’auto-registració

![HostMetadata Linux](img/HMD/HM3.png)

```bash
HostMetadata=Linux
```

Aquesta línia permet identificar el tipus de sistema i aplicar-hi accions automàtiques.

---

### Pas 4: Reiniciar el servei de l’agent

![Restart servei](img/HMD/HM4.png)

```bash
systemctl restart zabbix-agent.service
systemctl status zabbix-agent.service
```

Cal comprovar que el servei queda en estat **actiu**.

---

## Fase 2: Configuració del Descobriment

### Pas 1: Accedir a l’apartat de descobriment

![Menú descobriment](img/HMD/HM5.png)

* Anar a **Recopilació de dades → Descobriment**

Aquí es configuren les regles que permeten detectar equips a la xarxa.

---

### Pas 2: Crear la regla de descobriment

![Regla descobriment](img/HMD/HM6.png)

Configurar la regla amb els valors corresponents de xarxa, interval i tipus de comprovació.

Aquesta regla permet que Zabbix detecti automàticament els equips disponibles.

---

### Pas 3: Verificar que la regla està activada

![Regla activada](img/HMD/HM7.png)

Cal assegurar-se que la regla de descobriment està **activada** perquè pugui executar-se.

---

## Fase 3: Configuració de l’Auto-registració

### Pas 1: Crear una acció d’auto-registració

![Crear acció](img/HMD/HM8.png)

* Anar a **Alertes → Accions → Autoregistre**
* Crear una nova acció

Aquesta acció definirà què ha de fer Zabbix quan detecti un host nou.

---

### Pas 2: Definir la condició

![Condició Linux](img/HMD/HM9.png)

Configurar la condició perquè només afecti els hosts amb:

* **Metadatos de equipo**
* **contiene**
* **Linux**

---

### Pas 3: Confirmar la condició creada

![Condició creada](img/HMD/HM10.png)

Comprovar que la condició queda afegida correctament dins l’acció.

---

## Fase 4: Configuració de les Operacions

### Pas 1: Definir les operacions automàtiques

![Operacions](img/HMD/HM11.png)

Afegir les accions necessàries perquè Zabbix configuri automàticament el host:

* Afegir el host
* Afegir-lo a un grup
* Assignar plantilles

---

### Pas 2: Verificar l’acció creada

![Acció creada](img/HMD/HM12.png)

Cal revisar que l’acció estigui correctament configurada i **activada**.

## Fase 5: Verificació del Funcionament

### Pas 1: Comprovar els hosts descoberts

![Host descobert](img/HMD/HM13.png)

En aquest apartat es poden veure els dispositius detectats automàticament per la regla de descobriment.

### Pas 2: Verificar que el host s’ha registrat

![Host a Zabbix](img/HMD/HM14.png)

El host ja hauria d’aparèixer a la llista d’equips de Zabbix amb la configuració aplicada automàticament.

### Pas 3: Revisar els logs de l’agent

![Logs agent](img/HMD/HM15.png)

```bash

```

Això permet comprovar si l’agent s’està comunicant correctament amb el servidor.

## Fase 6: Gestió i Eliminació del Host

### Pas 1: Accedir a la llista d’equips

![Host a Zabbix](img/HMD/HM14.png)

* Anar a **Recopilació de dades → Equips**

Aquí es mostren tots els hosts registrats.

---

### Pas 2: Seleccionar el dispositiu

* Marcar el host que es vol eliminar

---

### Pas 3: Eliminar el host

* Fer clic a **Eliminar**
* Confirmar l’acció

Un cop eliminat, el dispositiu desapareixerà de la llista.

---

## Important: Evitar que el host es torni a registrar

Si l’auto-registració continua activa, el dispositiu pot tornar a aparèixer.

### Opció 1: Aturar l’agent

```bash
systemctl stop zabbix-agent
```

### Opció 2: Eliminar o comentar el `HostMetadata`

```bash
nano /etc/zabbix/zabbix_agentd.conf
```

Eliminar o comentar aquesta línia:

```bash
HostMetadata=Linux
```

Després, reiniciar el servei:

```bash
systemctl restart zabbix-agent
```
